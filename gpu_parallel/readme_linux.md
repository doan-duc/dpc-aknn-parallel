# DPC-AKNN GPU (Linux)

### 1. Biên dịch (Compile)
Mở terminal tại thư mục này và chạy:
```bash
# Dọn dẹp các file build cũ
make clean

# Biên dịch chương trình
make
```

### 2. Chạy chương trình (Run)

**Cách 1: Chạy bằng cấu hình mặc định (được thiết lập trong file `config.h`)**
```bash
./dpc_aknn_gpu
```
